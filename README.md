# myvim_config
A useful vim config

# Installation

Before running this script, make sure to install **vim** and **git**

Open the terminal and run these commands :

<ol>
    <li>Executive access to the script <ul><li>
        <code> chmod +x config.sh</code>
    </li></ul></li>
    <li>Run the configure script  <ul><li>
        <code> ./config.sh</code>
    </li></ul></li>
</ol>
